# 03-推荐资源

> 精选 AI 学习资源，分为入门、进阶、学术三个方向。每个资源附一句话推荐理由，帮你快速筛选。

---

## 入门方向

### 书籍

| 书名 | 作者 | 一句话推荐 |
|------|------|-----------|
| **《这就是ChatGPT》** | Stephen Wolfram | 用极简方式解释 LLM 内部原理，适合完全零基础的读者建立直觉 |
| **《Deep Learning》（花书）** | Ian Goodfellow 等 | 深度学习经典教材，前三章读完就能对 ML 有系统认知 |
| **《Attention Is All You Need》（论文）** | Vaswani 等 | Transformer 原始论文，所有现代 LLM 的基础，值得精读一次 |
| **《Prompt Engineering Guide》** | DAIR.AI | 免费的 prompt 工程入门指南，覆盖零样本、少样本、思维链等核心技巧 |

## 进阶方向

### 书籍

| 书名 | 作者 | 一句话推荐 |
|------|------|-----------|
| **《Speech and Language Processing》** | Daniel Jurafsky & James Martin | NLP 领域圣经级教科书，免费在线，内容极其全面 |
| **《Designing Machine Learning Systems》** | Chip Huyen | 从系统设计角度讲 ML 工程，离线评估、数据工程、部署监测等 |
| **《Building LLM Apps》** | Valentina Alto | LLM 应用开发实战，RAG、Agent、Fine-tuning 覆盖完整 |
| **《Natural Language Processing with Transformers》** | Lewis Tunstall 等 | 用 Hugging Face 生态做 NLP 项目，代码驱动，实战性强 |

| **OpenAI Function Calling 文档** | Function Calling 官方设计指南，理解和实现 Tool Use 必读 |

## 学术方向

### 必读论文清单

| 论文 | 一句话推荐 |
|------|-----------|
| **Attention Is All You Need** (Vaswani et al., 2017) | Transformer 的原始论文，后续所有 LLM 的原点 |
| **BERT: Pre-training of Deep Bidirectional Transformers** (Devlin et al., 2019) | 双向预训练的开创工作，NLP 进入预训练时代 |
| **GPT-3: Language Models are Few-Shot Learners** (Brown et al., 2020) | 大规模 in-context learning 的提出，few-shot prompt 范式确立 |
| **Training Language Models to Follow Instructions** (Ouyang et al., 2022) | InstructGPT 论文，RLHF 的里程碑工作 |
| **Constitutional AI** (Bai et al., 2022) | Anthropic 提出的无监督对齐方法，RLHF 之外的新路径 |
| **ReAct: Synergizing Reasoning and Acting** (Yao et al., 2022) | Agent 范式的核心论文，Reasoning + Acting 框架 |
| **Retrieval-Augmented Generation** (Lewis et al., 2020) | RAG 的原始论文，知识增强生成的基础框架 |
| **LLaMA: Open and Efficient Foundation Models** (Touvron et al., 2023) | Meta 开源系列的开端，开源模型的里程碑 |
| **DeepSeek-R1: Incentivizing Reasoning** (DeepSeek, 2025) | 推理模型的代表，纯 RL 训练方法的突破 |
