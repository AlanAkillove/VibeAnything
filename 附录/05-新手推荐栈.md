# 05-新手推荐栈

> 如果你是零基础起步的国内新手，这页给你一个经过验证的推荐组合：**不需要 VPN、不需要国际信用卡、0 成本**，每一步都有具体的操作指引。

---

## 首选组合：DeepSeek API + SiliconFlow + Kimi Web

| 工具 | 作用 | 费用 |
|------|------|------|
| **DeepSeek API** | 主力 LLM，编码和问答最强 | 极低价格，注册送 500 万 token，用几个月没问题 |
| **SiliconFlow API** | 模型聚合后备，跑其他开源模型 | 注册送 14 元额度 + 部分免费模型 |
| **Kimi Web Chat** | 长文档阅读和免费 Chat | 完全免费 |
| **VS Code** | 代码编辑器 | 免费 |
| **GitHub** | 代码托管和 Copilot 免费版 | 免费 |

---

## 为什么是这个组合

### DeepSeek V4 Flash（主力）
DeepSeek 是目前国产模型中性价比最高的选择：推理能力强（代码、数学、逻辑）、中文支持好、价格极低（百万 token 几毛钱）。Flash 版本适合日常编码和问答，Pro 版本适合复杂推理任务。

### SiliconFlow（后备 + 模型切换）
SiliconFlow 内置了大量开源模型（Qwen、GLM、Llama、DeepSeek 全系列），完全兼容 OpenAI 的 SDK，让你只用改 ase_url 和 model 就能切换。当 DeepSeek 偶有不稳定或你想对比不同模型的效果时，这是最佳替补。

### Kimi（免费 Chat + 长文档阅读）
Kimi 的 Web 端完全免费，200 万字上下文窗口，适合丢论文、长文档、代码库进去问问题。不需要开通任何付费，注册即用。

4. **VS Code**：打开 [code.visualstudio.com](https://code.visualstudio.com) — 下载安装
5. **GitHub**：打开 [github.com](https://github.com) — 邮箱注册

## 第二步：项目脚手架

创建一个新目录，初始化 Python 项目：

`powershell
mkdir my-ai-project
cd my-ai-project
python -m venv .venv
.venv\Scripts\activate
pip install openai python-dotenv
`

新建 .env 文件存放 API Key：

`env
DEEPSEEK_API_KEY=sk-你申请的key
SILICONFLOW_API_KEY=sk-你申请的key
`

新建 config.py 统一管理客户端：

`python
import os
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()

deepseek_client = OpenAI(
    api_key=os.getenv("DEEPSEEK_API_KEY"),
    base_url="https://api.deepseek.com"
)

siliconflow_client = OpenAI(
    api_key=os.getenv("SILICONFLOW_API_KEY"),
    base_url="https://api.siliconflow.cn/v1"
)
`

---

## 第三步：第一个调用

新建 hello.py：

`python
from config import deepseek_client

response = deepseek_client.chat.completions.create(
    model="deepseek-v4-flash",
    messages=[
        {"role": "system", "content": "你是我的 AI 编程助手，用中文回答问题。"},
        {"role": "user", "content": "用 Python 写一个读取 CSV 文件并返回统计摘要的函数"}
    ]
)

print(response.choices[0].message.content)
`

运行：

`powershell
python hello.py
`

看到输出就说明一切正常。把这个模板保存下来，以后换模型只需要改 model 和 client。

## 第四步：日常使用模式

`markdown
| 场景 | 用什么 | 为什么 |
|------|-------|--------|
| 写代码、debug、做项目 | DeepSeek V4 Flash API | 推理能力强，价格极低 |
| 长文档阅读（论文/代码库） | Kimi Web Chat 拖入文件 | 200 万上下文，完全免费 |
| 对比不同模型效果 | SiliconFlow 切换模型 | 同一个 SDK 改个 model 名就能换 |
| 日常聊天、搜索整合 | Kimi / DeepSeek Web | 免费，不需要 API 调用 |
| 有 VPN 条件时尝鲜 | Claude Sonnet 4 / GPT-4o | 国际旗舰模型，做复杂编码时可临时用 |
`

## 进阶路线（当 0 成本组合不够用时）

| 阶段 | 动作 | 成本 |
|------|------|------|
| 有少量预算 | 给 DeepSeek 充 50 元 — 够用几个月 | ~50 元 |
| 需要更强编码能力 | 开通 Claude API 或 Cursor Pro | ~/月 |
| 需要本地运行模型 | 装 Ollama，下载 Qwen2.5-7B 或 DeepSeek-R1 蒸馏版 | 需要显卡 |
| 需要团队协作 | 用 Dify 搭建 LLM 应用工作流 | 自部署免费，云端有免费额度 |
| 需要生产级监控 | 接入 Langfuse 做 trace 和 token 统计 | 有免费额度 |
