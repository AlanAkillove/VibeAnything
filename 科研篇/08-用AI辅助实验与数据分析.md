# 08-用 AI 辅助实验与数据分析

> 读完本文你将了解：如何用 AI 生成实验脚本、处理数据、可视化结果，以及如何用 AI 辅助写论文中的实验部分——包括实验设置、结果描述和分析。

---

## 你大概率遇到过的问题

实验和数据分析阶段是最消耗精力但 AI 最擅长辅助的环节：

- **跑实验的 boilerplate code 重复写**——每次新项目都写一遍一样的 dataloader、trainer、evaluator
- **数据处理脚本写得慢**——数据清洗、预处理、转换，大部分时间花在写 pandas 上
- **结果可视化审美堪忧**——Matplotlib 默认图又丑又难调
- **写论文实验部分时不知道怎么说**——"我们的方法比 baseline 好"一句话写完了，不知道该怎么展开分析
- **面对一堆实验数据不知道怎么分析**——不知道从哪些角度解读

这些痛点的共同点：**AI 在「写样板代码」和「生成初稿」两个环节效率极高，你只需要做校验和决策**。

---

## 常见的误解

| 你以为的 | 实际的 |
|---------|--------|
| AI 生成的实验代码可以直接跑 | 必须逐行检查——AI 经常产生微妙的 bug（如 tensor shape 对不上） |
| AI 能自动分析你的实验结果 | AI 只能帮你组织和描述结果，真正的分析（为什么涨点、为什么掉点）要你来判断 |
| AI 画图就是调用 matplotlib | AI 可以帮你写出 good-looking 的代码，但理解图的含义是人的工作 |
| AI 写实验部分等于它可以代替你做实验 | AI 写的是实验部分的初稿，实验设计和结果解读必须自己来 |

## AI 生成实验脚本

### 训练代码模板

```python
# 给 AI 的 prompt
prompt = """
帮我写一个 PyTorch 训练脚本，要求：
- 模型: ResNet-50（pre-trained on ImageNet）
- 数据集: CIFAR-100
- 优化器: AdamW, lr=1e-4, weight_decay=0.01
- 学习率: cosine annealing
- 训练 epoch: 100
- 评估: top-1 accuracy
- 使用 torch.cuda.amp 做混合精度训练
- log 到 wandb
"""
```

AI 生成的代码要检查：

- 数据预处理是否正确（Normalization 的 mean/std）
- Device placement 是否正确（model.to(device), input.to(device)）
- train/val 切分是否正确
- Grad scaler 是否正确使用

### 实验脚本生成

多个实验需要不同的配置组合，让 AI 帮你写 batch 脚本：

```bash
# AI 生成：自动扫 learning rate
for lr in 1e-4 5e-4 1e-3; do
  for wd in 0.001 0.01; do
    python train.py --lr $lr --weight_decay $wd --seed 42
  done
done
```

## AI 辅助数据处理

### 数据清洗

```python
prompt = """
我有这样一个 DataFrame，列名为：review_text, rating, date, user_id。
请帮我写数据清洗代码：
- 删除 rating 为空的行
- 去除 review_text 中的 HTML tag 和特殊字符
- 标准化日期格式（目前格式不统一）
- 去除重复的 user_id 行（保留 rating 最高的那一行）
"""
```

### 数据增强

```python
prompt = """
帮我写一段 PyTorch 的数据增强 pipeline：
- 随机水平翻转
- 随机旋转，最大 15 度
- RandomResizedCrop 到 224x224
- ColorJitter: brightness=0.2, contrast=0.2
- 最后 Normalization（ImageNet 的 mean/std）
"""
```

## AI 辅助结果可视化

### 生成 publication-ready 图表

```python
prompt = """
我有一组实验结果，请帮我画图：

实验结果（acc %）：
- Ours: 87.3
- Baseline A: 82.1
- Baseline B: 84.5
- Ours (w/o attention): 85.2

要求：
1. 画 bar chart
2. 柱子上标注数值
3. 使用 seaborn 的 muted 配色
4. 字体 12pt
5. 保存为 PDF，分辨率 300dpi
6. 标题：Test Accuracy Comparison
"""
```

### 结果对比图

```python
prompt = """
我有 3 个模型的训练 loss curve（每个有 100 个 epoch 的 loss），
数据在 loss_logs.csv 中，columns: epoch, ours_loss, baseline1_loss, baseline2_loss。

请帮我画：
1. 三条 loss 曲线在一张图上
2. 用不同的颜色和线型区分
3. 加 legend
4. 合理设置 y 轴范围（不要从 0 开始，展示有价值的部分）
5. 字体大小 11pt
6. 保存为 PDF
"""
```

### 典型的 publication-ready 图类型

| 用途 | 图表类型 | 工具 |
|------|---------|------|
| 模型对比 | Bar chart (with error bar) | seaborn / matplotlib |
| Loss curve | Line chart | matplotlib |
| Parameter sensitivity | Line chart (multi-line) | matplotlib |
| Correlation / scatter | Scatter plot | seaborn / plotly |
| Confusion matrix | Heatmap | seaborn |
| 分布对比 | Box plot / Violin plot | seaborn |

## AI 辅助写实验部分

论文实验部分通常包含这几个段落：

### 1. Experimental Setup

```markdown
## Experiments

### Experimental Setup
We conducted experiments on three datasets: ... 
We used ... as evaluation metrics.
All models were trained on ... with ... hyperparameters.
For each experiment, we report mean and standard deviation over 5 random seeds.
```

这个让 AI 帮你写初稿最省力——把你的配置列表丢给 AI 就好。

### 2. Main Results

```markdown
## Main Results

As shown in Table 1, our method outperforms all baselines across all datasets.
On Dataset A, we achieve 87.3% accuracy, which is a 2.8% improvement over the
previous SOTA (Baseline B, 84.5%). On Dataset B, ...
```

提供对比表格，让 AI 生成初稿。**但你要自己判断**：

- 为什么在这个数据集上提升大，在那个数据集上提升小？
- 哪些 baseline 表现出乎意料？为什么？
- 哪个结果最有说服力？放在最前面说。

### 3. Ablation Studies

```markdown
### Ablation Studies

To validate the contribution of each component, we conducted ...
Table 2 shows the results of removing each component.

When removing the attention module, accuracy drops by 3.1%, indicating ...
The residual connections are essential, and removing them leads to ...
```

Ablation study 的描述最适合让 AI 写——因为结构固定、描述模式单一。

## AI 辅助数据分析的提示

**让 AI 分析你给的数值，而不是凭空建议：**

```
❌ "帮我分析一下实验结果通常从哪些角度分析"（太泛了）
✅ "我的实验结果如下表，请帮我把最显著的发现写成一个段落：..."
```

**给 AI 的 prompt 要结构化：**

```markdown
我的实验结果（accuracy in %，mean +/- std over 5 seeds）：

Ours: 87.3 +/- 0.4
Baseline A: 82.1 +/- 1.2
Baseline B: 84.5 +/- 0.8
Ours (w/o attention): 85.2 +/- 0.6

请帮我：
1. 指出最显著的结果差异
2. 分析为什么 std 最小的反而是我们的方法
3. 生成一段 Result 部分的英文初稿（200 words）
```

## 要点总结

1. **AI 最擅长写实验 boilerplate**：训练脚本、dataloader、数据预处理，生成后手动检查
2. **数据清洗和可视化交给 AI**：pandas 操作和出版级绘图代码都是 AI 的高效区
3. **实验部分的初稿让 AI 写**：Experimental Setup、Main Results、Ablation Studies 三个段落的模板化程度高，AI 写初稿你改
4. **给 AI 的数据要具体**：直接丢数值和表格，让 AI 做描述性的总结和分析
5. **真正的实验分析自己来**：为什么涨点、为什么掉点、实验结果有什么 insight——这些是 AI 做不到的
6. **所有 AI 生成的代码和文本都要人工验证**：形状对齐、数值正确性、叙述的准确性，一个都不能省
