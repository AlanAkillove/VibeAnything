# 贡献指南

## 如何贡献

1. Fork 本仓库
2. 创建你的分支：`git checkout -b feature/你的改动`
3. 提交改动：`git commit -m "feat: 你的改动说明"`
4. 推送到分支：`git push origin feature/你的改动`
5. 提交 Pull Request

## 写作规范

参见 `docs/superpowers/specs/2026-06-18-vibe-anything-tutorial-design.md` 中的写作规范（5. 写作规范 v2 — 问题驱动）。

核心要点：
- 每篇文章以真实问题场景开场
- 包含常见误解和分步解决方案
- 中英夹杂，术语保留英文
- 不出现 em dash（——），用中文标点替代
- 控制文章长度在 1,500-3,000 字
- 每篇文章由 AI agent review 后合并
