# VibeAnything

> 一个面向 CS 本科生的 Vibe Coding 入门教程

本项目帮助计算机专业本科生系统性地学习如何在 Vibe Coding 时代利用 AI 提升开发效率，涵盖从 LLM 基础原理到动手实践的完整路径。

## 写作原则

- **问题驱动**：每篇文章从一个实际场景出发，给出可以跟着做的解决方案
- **中英夹杂**：术语保留英文，解释用中文
- **每篇有 review**：所有内容都经过 AI agent 审查，确保准确性
- **调研在先**：涉及具体工具和模型时，先查阅官方文档再撰写
