# VibeAnything

> 一个面向 CS 本科生的 Vibe Coding 入门教程（68 篇文章 · 16 个模块）

本项目帮助计算机专业本科生系统性地学习如何在 Vibe Coding 时代利用 AI 提升开发效率，涵盖从 LLM 基础原理到动手实践的完整路径。

不确定从哪里开始？参考 [入门导航.md](./入门导航.md) 根据你的目标选择推荐路径。

## 阅读建议

- **完全新手**：从 **基石篇** 开始，系统建立 LLM 认知
- **想先动手**：从 **环境准备篇** → **工具篇** 快速上手
- **想系统学习**：按顺序通读 **基石篇 → 进阶篇 → 工作流篇 → AI协作心法篇**
- **关注就业**：参考 **中国大陆生态篇 + 行业篇**
- **不确定**：先看 [入门导航.md](./入门导航.md)

## 写作原则

- **问题驱动**：每篇文章从一个实际场景出发，给出可以跟着做的解决方案
- **中英夹杂**：术语保留英文，解释用中文
- **时效优先**：所有模型版本号截至 2026 年 6 月，标注具体参照
- **调研在先**：涉及具体工具和模型时，先查阅官方文档再撰写
- **可操作性**：每篇文章末尾有"更进一步"章节，指向相关阅读
